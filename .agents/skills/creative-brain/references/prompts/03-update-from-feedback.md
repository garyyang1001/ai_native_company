# Update From Feedback

Use this whenever the user reacts to the creative brain or to an output made from it.

## Feedback labels

Ask the user to label:

```txt
Keep:
Cut:
Rewrite around:
More like:
Less like:
```

## Update process

1. Identify what changed.
2. Update the relevant brain files.
3. Add a dated note to `feedback-memory.md`.
4. Preserve approved phrasing.
5. Add rejected patterns to anti-slop rules.
6. Summarize what the agent should do differently next time.

## Feedback memory format

```md
## YYYY-MM-DD — Feedback update

### Keep

### Cut

### More like

### Less like

### Rules learned

### Files updated
```
