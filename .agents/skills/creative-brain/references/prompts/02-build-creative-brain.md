# Build Creative Brain

After the interview, create files under:

```txt
outputs/creative-brain/
```

Use these files:

- `README.md`
- `identity.md`
- `voice.md`
- `taste.md`
- `anti-slop.md`
- `references.md`
- `content-system.md`
- `ideas.md`
- `current-state.md`
- `feedback-memory.md`

## Build rules

- Use the user's actual words where possible.
- Separate facts from guesses.
- If something is uncertain, label it as uncertain.
- Do not over-flatter the user.
- Do not turn them into a generic founder/creator.
- Make each file useful for a future agent.

## Final response after draft

After creating the files, say:

```txt
built the first version of your creative brain.

quick read:
- [3-5 bullets summarizing what the brain understands]

now tell me:
Keep:
Cut:
Rewrite around:
More like:
Less like:
What feels wrong?
What feels most like you?
```
